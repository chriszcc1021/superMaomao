# 喵喵远征 美术制作清单

## 目标

当前项目的核心玩法已经可以跑通，但大部分画面仍处于程序占位阶段。  
本清单用于指导第一轮正式美术替换，优先把项目从“程序占位版”提升到“可展示、可试玩、可继续打磨”的状态。

原则：
- 先做高频可见资源，再做低频资源
- 先做可替换的一轮正式稿，再做终稿精修
- 先保证风格统一，再追求细节复杂度

---

## 优先级说明

- `P0`：必须优先做，直接影响游戏观感
- `P1`：建议第一轮同步做，能明显提升完成度
- `P2`：可以放到第二轮，不影响当前版本展示

---

## P0 核心资源

### 1. 营地建筑套件

需要建筑：
- 猫窝 `cat_house`
- 粮仓 `granary`
- 猫粮田 `food_farm`
- 金矿 `gold_mine`
- 产房 `nursery`
- 医院 `hospital`
- 爱心猫窝 `heart_cat_house`
- 墓地 `cemetery`
- 招财猫神龛 `fortune_cat`

每个建筑建议至少提供：
- 已建造状态
- 未解锁/半透明占位状态
- 被点击/选中时的高亮表现

建议风格：
- 等距视角
- 形体清晰，便于一眼分辨建筑功能
- 色彩区分明显，避免建筑之间过于相似

### 2. 营地地面与环境

需要内容：
- 等距地面基础贴图
- 草地/土路/石板等分区
- 建筑下方阴影
- 基础环境装饰，如木箱、围栏、草丛、路牌、灯笼

目标：
- 让营地从“空白摆件场”变成一个真实的经营空间

### 3. 猫咪基础形象

需要先做统一模板，再做差异化外观。

建议至少覆盖：
- 营地待机
- 营地走动
- 睡觉
- 繁育中
- 祈祷/招财
- 挖矿
- 治疗
- 生病
- 死亡

建议优先做的品种差异：
- 虎斑
- 布偶
- 暹罗
- 橘猫
- 黑猫
- 英短

### 4. 战斗猫形象

需要内容：
- 俯视视角战斗猫
- 待机
- 移动
- 攻击
- 受击
- 死亡

备注：
- 当前战斗中已经有固定血条和经验条显示
- 美术需要预留头顶 UI 可见空间

### 5. 敌人套件

需要敌人：
- 小猴兵 `small_monkey`
- 石猴 `stone_monkey`
- 坦克猩猩 `tank_gorilla`
- 精英猴 `elite_monkey`
- Boss 猩猩王 `boss_gorilla_king`

建议至少提供：
- 基础战斗态
- 受击表现
- 死亡表现

### 6. 战斗地图

当前战斗地图仍是占位网格。

第一轮建议先做：
- 一套基础地面风格
- 可循环使用的地图底图
- 简单环境元素，如石块、草、树根、遗迹碎片

目标：
- 把战斗场景从程序测试图替换成正式可展示地图

### 7. UI 基础皮肤

优先覆盖：
- 营地 HUD
- 流浪猫来访面板
- 繁育界面
- 远征选择页
- 战斗 HUD
- 选卡面板

建议先统一：
- 面板底板
- 标题条
- 按钮样式
- 进度条样式
- 图标框样式

## P1 完成度提升资源

### 8. 进度条资源

需要统一风格的：
- 血条
- 经验条
- 猫粮/金币等资源条

### 9. 建筑工作反馈特效

当前已有逻辑和漂字提示，建议补正式表现：
- 猫粮田：`猫粮 +1`
- 金矿：`金币 +1`
- 招财猫：`金币 +1`

建议做法：
- 小图标 + 数字
- 轻量向上飘动
- 不遮挡猫和建筑主体

### 10. 战斗特效

建议第一轮先做轻量版：
- 普通命中
- 暴击命中
- 回血
- 升级
- 无敌状态闪烁
- 敌人死亡反馈

### 11. 图标系统

建议统一一套图标风格：
- 资源图标
- 建筑图标
- 职业图标
- 品种图标
- 状态图标

### 12. 远征节点图标

需要：
- 普通战斗
- 精英战斗
- Boss
- 商店
- 问号事件

目标：
- 远征地图节点一眼可辨识

### 13. 卡牌视觉样式

建议第一轮先做：
- 卡框
- 稀有度边框
- 背景底图
- 图标占位

不一定要第一轮就把每张卡插画画满。
