## 一、游戏定位 & 大循环

### 定位

**营地建设养成**（咩咩启示录风格）× **肉鸽远征**（分叉路线 6 层）× **幸存者战斗**（单猫操控，全自动武器）× **基因繁育**（少而精，技能血脉积累）

### 大循环

```
【营地】每日建设/繁育/管理猫咪
     ↓
选一只成年猫出征（一生限一次）
     ↓
【远征】肉鸽地图 6 层，选路推进
     ↓
【战斗】幸存者式，90秒/场，升级选卡牌
     ↓
升到 Lv5/10/15 → 习得技能写入基因槽
     ↓
远征结束 → 猫回营地，标记"已出征"
     ↓
猫继续在营地工作/繁育（直到老年期）
     ↓
子代继承父母基因槽内容 → 带出征
     ↓
循环，血脉越来越强
```

### 3.4 外观基因（8 槽，8640 种组合）

| 基因槽 | 选项 | 影响属性 |
|--------|------|---------|
| head | round/sharp/wide | HP（200/130/300） |
| ear | upright/fold/big/hairless | 攻击范围（3/2/4/2.5格） |
| eyeColor | blue/green/amber | 暴击率（5%/10%/18%） |
| eyeShape | round/narrow | 暴击倍率（1.5x/2.2x） |
| furMain | orange/black/white/gray | 攻击力（12/16/9/11） |
| furAccent | white/tan/none | 金币倍率（1.1/1.2/1.0） |
| pattern | none/tabby/tortoise/spotted/colorpoint | 初始被动技能倾向 |
| tail | long/short/curl/bobtail | 攻速（0.8/1.4/1.1/1.0次/秒） |

### 3.5 特殊基因槽（3个，统一共用）⭐

每只猫有 **3个特殊基因槽**，三种基因类型共用同一套槽位：

| 基因类型 | 说明 | 获取方式 |
|---------|------|---------|
| **主动技能基因** | 战斗中触发的技能 | 远征升级Lv5/10/15习得，或触发隐藏条件 |
| **战斗被动基因** | 战斗中生效的天赋 | 出生继承，繁育遗传 |
| **营地被动基因** | 营地日常中生效 | 出生继承，繁育遗传 |

```
示例：猫咪「橘橘」的3个基因槽
├── 槽1：主动技能基因「越打越来劲」
├── 槽2：战斗被动基因「韧性体质」
└── 槽3：营地被动基因「勤劳小能手」
```

**关键取舍：** 放营地被动 = 少一个战斗基因；3个槽全是战斗基因 = 营地效率低。

**出生时：** 通过繁育从父母3个槽中各随机继承（40-45%概率），空槽留白。
**远征后：** 习得技能写入空槽；若3槽已满，玩家选择替换哪个。

**繁育时的基因遗传：**
- 子代每个槽位内容：45%父 / 45%母 / 10%突变（随机一个基因池内的基因）
- 每个槽独立计算，不保证全部遗传

### 3.6 卡牌 vs 技能：关键区分

| | 卡牌 | 技能（主动技能基因） |

### 3.10 猫咪生命周期

| 阶段 | 天数 | 状态 |
|------|------|------|
| 幼猫期 | 3天 | 不可工作/出征，消耗1猫粮/天 |
| 成年期 | **最长30天** | 可工作、可出征一次（一生限一次）、可繁育（上限3次） |
| 老年期触发 | 30天满 **或** 繁育满3次（以先到者为准） | — |
| 老年期 | 5天 | 当前代码版本中可正常工作并获得建筑经验，也可继续繁育；远征限制以后续平衡为准 |
| 自然死亡 | 老年期结束 | 写入墓地记录 |

**设计意图：**
- 出征时机的决策：早出征技能早写入基因，还有时间繁育；晚出征Build更强但繁育期缩短
- 时间压力：成年期30天倒计时，玩家不能无限拖延

### 3.11 繁育规则（少而精）

- 每只猫最多参与繁育 **3次**（计入成年期30天老年触发条件）
- 繁育**不消耗金币**，但必须先建造产房
- 产房建成后：单次繁育成功概率 **0.65**
- 营地猫口上限：**猫窝格数**（初始5格，最大10格）

### 3.12 战斗影响占比

```
猫本身属性：30%
  ├── 基础数值（攻击/血量/攻速）：10%
  ├── 职业基底：                  10%
  └── 战斗被动基因（天赋）：      10%

卡牌 Build：70%
  ├── 武器卡（主动技能）：        35%
  └── Buff 卡（属性倍率）：       35%
```

## 四、远征系统

### 4.1 出征

**每次出征 1 只猫**，一生限出征一次，出征后标记"已出征"，不可再次出征。

出征条件：猫为成年状态、健康（非生病/濒死）、未曾出征。

### 4.2 地图结构（分叉路线）

```
[起点] → 层1选路 → 层2选路 → 层3选路 → 层4选路 → 层5选路 → [Boss层]
每层 2-3 条路可选
```

- 总层数：**6 层**，最后一层固定 Boss 战
- 一次完整远征约 **10 场战斗**

### 4.3 节点类型

| 节点 | 内容 |
|------|------|
| ⚔️ 普通战斗 | 90秒，存活即胜利，击杀积累小鱼干升级选卡 |
| 💀 精英战斗 | 120-180秒，含精英猴，奖励更高稀有度卡牌 |
| 👑 Boss 战 | 无时限，消灭Boss（设计目标2-3分钟） |
| ❓ 问号事件 | 随机触发一种（见4.4） |
| 🛒 商店 | 以卡换卡（见4.5） |

### 4.4 节点概率配置（每层固定生成2个节点，玩家**二选一**）

> **设计原则：** 每层固定呈现2个节点选项，玩家选一个进入，另一个放弃。两个节点类型尽量不同（系统会重新随机第二个以避免完全重复）。

| 层数 | 普通战⚔️ | 精英战💀 | 问号❓ | 商店🛒 |
|------|---------|---------|-------|-------|
| 第1层 | 70% | 0% | 20% | 10% |
| 第2层 | 55% | 15% | 20% | 10% |
| 第3层 | 45% | 20% | 20% | 15% |
| 第4层 | 35% | 30% | 20% | 15% |
| 第5层 | 25% | 40% | 20% | 15% |
| 第6层 | Boss固定，无二选一 | — | — | — |

### 4.5 问号事件（❓）

| 概率 | 事件 | 内容 |
|------|------|------|
| 30% | 🪙 意外收获 | 获得金币 15-35 |
| 25% | 🎁 神秘祝福 | 随机获得一个本次远征有效的 Buff |
| 20% | 🐱 流浪幼崽 | 发现一只流浪猫，加入营地等待队列 |
| 15% | ⚡ 小麻烦 | 猫HP上限-5%（本次远征） |
| 10% | 📜 奇遇故事 | 纯文字剧情，带小惊喜或小坏事 |

> 流浪猫来源：营地日常（基础38%/日）+ 问号事件，共用同一等待队列（最多3只）。

### 4.6 商店（以卡换卡）

- **换卡**：背包里选1张卡，换候选池中1张新卡（每个商店节点限换1次）
- **扩格**：用3张卡换1个背包格（背包最多至10格）

### 4.7 远征结算

- **成功：** 金币 = `50 × 战斗胜利场数`
- **失败：** 金币 = `20 × 战斗胜利场数`
- 主动技能基因无论成败均写入基因槽

## 五、战斗系统（幸存者式）

### 5.1 战斗形式

```
玩家手动控制猫移动（WASD / 摇杆）
所有武器全部自动触发
鼠标/摇杆方向 = 攻击优先朝向
  · 有方向输入 → 优先攻击鼠标指向区域的敌人
  · 无方向输入 → 默认攻击最近敌人
击杀掉落小鱼干（局内经验，本场用完即弃）
升级 → 暂停 → 三选一卡牌
  · 第一次升级（Lv1→2）：三选一池子仅出武器卡，强制选择，不可跳过
  · Lv2+ 之后：恢复正常（武器卡 + Buff卡混合）
90秒存活（普通）/ 消灭目标（精英/Boss）→ 战斗胜利
猫HP归零 → 本场失败，远征继续但少一次胜利
```

### 5.2 基础爪击（所有猫通用）

> **设计意图：** 猫进入战场时没有武器卡，需要一个保底攻击手段以积累首升经验。

| 属性 | 数值 |
|------|------|
| **技能名** | 普通爪击 |
| **攻击形式** | 向鼠标方向发射单体爪影，直线飞行 |
| **伤害** | 攻击力 × 0.8 |
| **攻击间隔** | 1.0s（固定，不受猫的攻速属性影响） |
| **射程** | 3格 |

**规则：**
- 所有猫（无论职业、品种、基因）基础爪击完全一致
- 进入战场时**立即生效**，无需解锁
- **永久存在**，不可被替换、叠加或进化
- **不占武器卡背包格**（背包4格只算选的武器卡）

### 5.3 小鱼干掉落（局内经验）

| 敌人 | 掉落小鱼干 |
|------|-----------|
| 小猴兵 | 1 |
| 投石猴 | 2 |
| 猴子群（单只） | 1 |
| 坦克猩猩 | 5 |
| 毒猴 / 炸弹猴 | 2 |
| 精英敌人 | 15-25 |
| Boss | 50 |

### 5.4 升级经验曲线

> **设计目标：** 首次升级约在战斗开始 10 秒内达到，确保玩家尽快拿到第一张武器卡。

| 当前等级 | 升级所需小鱼干 | 累计小鱼干 |
|---------|-------------|----------|
| Lv1 → 2 | **5** | 5 |
| Lv2 → 3 | 15 | 20 |
| Lv3 → 4 | 25 | 45 |
| Lv4 → 5 | 35 | 80 |
| Lv5 → 6 | 45 | 125 |
| 之后每级 | +15 递增 | — |

**开场敌人密度（前15秒）：**

```
0s  → 立刻生成 4只小猴兵
5s  → 再生成 3只小猴兵
10s → 再生成 2只投石猴 + 3只小猴兵
```

## 十、视角与 UI 设计

### 10.0 双视角设计 ⭐

| 场景 | 视角 | 参考 |
|------|------|------|
| 营地 | **斜45°等距视角（Isometric / 3/4视角）** | 咩咩启示录 |
| 战斗 | **纯俯视角（Top-Down）** | 吸血鬼幸存者 |

场景切换流程：
```
营地（Isometric） → 出征 → 远征地图UI → 进入战斗（Top-Down）
                                        → 战斗结束 → 远征地图UI
                                                    → 远征结束 → 营地（Isometric）
```

**技术注意：**
- 营地场景需启用 **Y-Sort**，确保猫走到建筑后面时正确遮挡
- 营地素材需要斜角版本（建筑/猫咪均需等距角度 Sprite）
- 两个场景使用不同摄像机和渲染设置，不混用
- Godot 4 原生支持 Isometric Tilemap，可直接使用

### 10.1 营地主界面（参考咩咩启示录，等距视角）
- 斜45°等距视角，营地全貌可见，猫咪自由走动
- 顶部HUD：金币 / 猫粮 / 当前天数
- 点击建筑弹出交互菜单
- 流浪猫来访时门口弹出通知卡片

### 10.2 战斗界面（参考吸血鬼幸存者，Top-Down）
- 纯俯视角2D，摄像机跟随猫
- 左上：猫HP条 + 头像 + 职业图标
- 右上：层数 / 节点类型 / 计时器（普通战显示倒计时）
- 底部：升级经验条 + 已装备卡牌槽
- 升级时：全屏半透明遮罩 + 三张卡牌选择

### 10.3 繁育界面
- 左右两列：父猫/母猫展示
- 外观基因8槽：图标+颜色
- 特殊基因3槽：图标+类型标签（主动/战斗/营地）
- 中间：预测子代属性范围
- 下方：确认繁育按钮
